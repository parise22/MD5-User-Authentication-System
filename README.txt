Welcome To The FileSystem.

To Compile, Follow These Instructions.

- Download All Attached Files and Place Them In The Same Directory.
- Open a terminal or ssh client that supports C++11.
- Open The Directory.
- Compile With:
		
	CC main.cpp function.cpp -o FileSystem -std=c++11

- To Run For New User:

	./FileSystem -i

- To Run For Existing User:

	./FileSystem
